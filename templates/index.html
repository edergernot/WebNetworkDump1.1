A simple web-application to get infos out of networkdevices

This tools will discover IP-Networks and try to login via SSH with the provided credentials.

When discovery is done, you can do a dump and it will do a SSH-Session to every device and do a bunch of commands to get network status.

Data will be stored in files and you can download the files a singel zip file.
